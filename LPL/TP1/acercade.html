Resumen:

  • Las páginas web pueden ser estáticas o dinámicas.
  • Las páginas estáticas fueron las primeras en desarrollarse y aún se usan.
  • Son útiles para presentar datos en diversas situaciones.
  • Son fáciles de crear, instalar y mantener.
  • Las dinámicas permiten mayor interacción con el usuario.
  • Son necesarias para procesar datos ingresados en formularios u otras funciones interactivas.
